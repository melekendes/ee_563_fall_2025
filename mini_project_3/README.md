# EE 563 - Mini Project 3: Applying LLMs for Multiple Tasks

#This repository contains the scripts for the "LLM's with HuggingFace Experiments and Text to Image" project.

## Scripts

* `ai_proc.py`: Backend logic containing the `AIProcessor` class (OOP structure).
* `app.py`: Frontend user interface built with Gradio.
* `text_to_image.py`: Script for generating images and testing parameters.
* `compare.py`: Script for benchmarking LLM speeds.
